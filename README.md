# ssr-next

项目基于 Next.js 进行开发

## dependencies

- next.js
- react
- react-dom

## scripts for dev

```
"dev": "next",
"build": "next build",
"start": "next start"
```

`next.js` 提供了 `cli` 命令行工具 `next`

## 对 next.js 的解读

`next.js` 是一个应用框架，本身是基于 `node.js react express webpack babel just touch node-fetch` 等一众主流前端技术的整合，并且提供了配套的 `cli` 命令行工具，提供开箱即用的应用开发解决方案。
